## ğŸ® Usage Guide

### Creating a Study Room

1. **Log in** to your account
2. Click **"Create Room"** button
3. Fill in:
   - Room name
   - Topic (select from dropdown or create new)
   - Description
4. Click **"Create"** and start discussing!

### Joining Discussions

1. Browse rooms on the **home page**
2. Use **filters** to find specific topics
3. Click on a room to **enter**
4. Start **posting messages** and learning!

### Managing Your Profile

1. Click on your **username** (top right)
2. Select **"Profile"**
3. Update your:
   - Avatar
   - Bio
   - Social links
4. View your **activity history**
